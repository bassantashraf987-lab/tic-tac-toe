#Developed a desktop-based Tic Tac Toe game using Python and the Tkinter GUI library.

##About the game:

###The application provides an interactive two-player gaming experience with a user-friendly graphical interface. The game automatically detects winning combinations across rows, columns, and diagonals, highlights the winning tiles, handles tie scenarios, and includes a restart feature for seamless gameplay. The interface is centered on the screen and uses visual indicators such as color changes and status messages to enhance the user experience.

* Implemented turn-based logic for two players (X and O) with real-time turn tracking.
* Developed an interactive graphical user interface (GUI) with responsive button-based gameplay
* Created automated win detection for rows, columns, and diagonal combinations.
* Applied event-driven programming concepts using Tkinter button commands and callbacks.
